TECHNICAL NOTES

Color Space Considerations
  - iOS icon uses Display P3 (wider gamut)
  - Web assets should use sRGB
  - Print requires CMYK conversion
  - #FEFF00 will appear slightly different across color spaces
  - Document that Display P3 is canonical for iOS
  
File Format Recommendations
  Web: SVG for logos, PNG for photos/complex images
  iOS: PNG for raster assets, PDF for vectors
  Print: PDF or EPS with embedded fonts
  Social: PNG or JPG (RGB, sRGB color space)
  
Resolution Standards
  Web: 2x and 3x for Retina displays
  iOS: @2x and @3x asset catalogs
  Print: 300 DPI minimum
  Social: Platform-specific (usually 72 DPI acceptable)
